# movienight

<!-- badges: start -->
<!-- badges: end -->

Let R recommend a film after a long day of coding.

## Installation

You can install from GitHub

``` r
# install.packages("devtools")
devtools::install_github("nvietto/movienight")
```

## Example

``` r
library(movienight)
## movienight()
## [1] "One Flew Over the Cuckoo's Nest"
```

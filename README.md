# movienight

Let R recommend a film.

## Installation

You can install from GitHub

``` r
# install.packages("devtools")
devtools::install_github("nvietto/movienight")
```

## Example

``` r
library(movienight)

# then call movienight()

movienight()

#[1] "Whiplash"

```
